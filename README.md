# RadarDev-Front

## Scripts

Para executar o seu projeto você pode executar os comandos abaixo:

### `yarn start`

Executa a aplicação no modo desenvolvimento.
Abra o seu localhost.

```
http://localhost:3000
```

### `yarn test`

Inicia o teste no modo exibição interativa.

### `yarn build`

Faz o build da sua aplicação.
